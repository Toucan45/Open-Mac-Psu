# Open Mac Psu


_an open solution to replacing faulty power supplies in macintosh se and se/30s_





<img width="2374" height="1298" alt="A render of the Open Mac Psu with components added" src="https://github.com/Toucan45/Open-Mac-Psu/blob/42de62fb39882a683d26f4ef893aa421c86c92b6/images/OPENpsu34.png" />


## Gerbers
Gerbers can be found [here] or in the /gerber directory. You may also edit and generate gerbers however you may please with the KiCad files. <br>
I personally found that JLCPCB provided the cheapest prices as of 2026, but I encourage you to check multiple places.
